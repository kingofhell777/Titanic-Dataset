 Titanic Age Distribution Analysis 

This project performs a basic Exploratory Data Analysis on the AGE column of the Titanic dataset.  
It focuses on understanding the distribution of passenger ages using visualizations and statistical measures.

---

# Features

- Loads the Titanic dataset (`tested.csv`) using Pandas
- Cleans the `Age` column by removing missing values
- Plots:
  - Histogram with KDE (distribution curve)
  - Boxplot (outlier detection + spread)
- Calculates:
  - Mean
  - Median
  - Mode
  - 10% Trimmed Mean
  - Skewness
- Provides a short interpretation of the distribution shape
